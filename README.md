# Projeto InsightIA – Fundamentos de Gestão de Projetos

Este repositório contém o projeto acadêmico **InsightIA**, desenvolvido para a disciplina de Fundamentos de Gestão de Projetos
do Centro Universitário UniFECAF.

## Conteúdo
- Documento principal em formato Word (.docx) ajustado ao padrão ABNT
- Estrutura completa do projeto: introdução, desenvolvimento, artefatos e conclusão

## Objetivo
Demonstrar a aplicação prática dos conceitos de gestão de projetos no planejamento de um produto digital baseado em IA.
